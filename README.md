# studySpring
스프링 관련 강의

스프링 핵심 원리 고급편
============
- springCore_advanced - 로그 추적기, 쓰레드 로컬 - ThreadLocal, 템플릿 메서드 패턴과 콜백 패턴
- proxy - 프록시 패턴과 데코레이터 패턴  
  - 프록시 
    - 체인이 가능함  --> 역할 위임이 가능  
  - 데코레이터 패턴 : 원래 서버가 제공하는 기능에 더해서 부가 기능을 수행.  
  ex) 요청 값이나, 응답 값을 중간에 변형.  
  ex) 실행 시간을 측정해서 추가 로그를 남긴다.
